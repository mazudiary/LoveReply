# LoveReply

A beautiful, creative, and romantic web page designed to express love in a unique and interactive way.

## Features
- Animated glowing hearts and floating petals for a dreamy atmosphere
- Typewriter effect for romantic quotes
- Handwritten-style animated love note
- Interactive "Send Love" button with heart burst animation
- Soft background music (toggleable)
- Modern, elegant, and responsive design

## How to Use
1. Open `index.html` in your web browser.
2. Enjoy the interactive and romantic experience.
3. Click the music button (top right) to play/pause background music.
4. Click the "Send Love" button to see a heart burst animation.

## Customization
- You can edit the quotes, replies, and love note in `index.html` to personalize your message.
- Change colors, fonts, or animations in the `<style>` section for a different look.
